# 💍 Wedding Invitation Website

A beautiful and responsive Wedding Invitation Website created using HTML, CSS, and JavaScript. This website allows couples to invite guests digitally with elegant animations, countdown timer, background music, event details, gallery, and venue information.

## ✨ Features

* Beautiful wedding-themed design
* Responsive layout for mobile and desktop
* Background music support
* Countdown timer to wedding day
* Bride & Groom introduction section
* Love story timeline
* Wedding gallery
* Wedding event details
* Venue location with Google Maps link
* Smooth scrolling navigation
* Modern and elegant UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🚀 How to Run

1. Download or clone the repository.
2. Place your music file as `music.mp3`.
3. Customize names, dates, photos, and venue details.
4. Open `index.html` in your browser.

## ❤️ Customization

You can easily customize:

* Bride & Groom names
* Wedding date
* Gallery images
* Event schedule
* Venue location

* Colors and fonts
